## 처리 원칙

- Lab Note는 기본 생성하지 않았다.
- 모든 모듈 파일에는 최종 Coverage Map, 최종 분리 후보, 확인 필요 항목을 포함했다.
- PDF가 일반 텍스트로 추출되지 않아 OCR을 사용했다.
- OCR 오인식 가능성이 있는 고유명사, 수치, URL, 명령어는 Concept Wiki 작성 시 원본 PDF 렌더와 재대조해야 한다.
