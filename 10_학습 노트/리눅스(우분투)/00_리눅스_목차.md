## 실습 자료

- `쉘 스크립트/` - 서버 구성 스크립트와 설정 파일. 현재 운영 기준이 아니라 학습 source/lab asset으로 본다.

## 현재 재시작 지점

- Ubuntu 기본 조작과 서버 운영 개념은 [[10_학습 노트/리눅스(우분투)/우분투|우분투]]에서 시작한다.
- 웹/DB 서버 구성 흐름은 [[10_학습 노트/리눅스(우분투)/웹,DB서버 스크립트 파일|웹,DB서버 스크립트 파일]]을 먼저 보고, 실제 스크립트는 `쉘 스크립트/`에서 확인한다.
- 종합 복습이나 시험형 문제 흐름은 [[10_학습 노트/리눅스(우분투)/마지막 총정리 문제|마지막 총정리 문제]]를 본다.

## RAW / legacy 주의

- `쉘 스크립트/`의 `.sh`와 `.txt` 파일은 당시 실습 환경 기준 source다. 현재 환경에 그대로 적용하지 않는다.
- `우분투`와 `마지막 총정리 문제`는 큰 덩어리 노트다. 자주 다시 찾는 주제는 나중에 명령어 노트나 concept/lab note로 분리 후보가 된다.
