## 현재 재시작 지점

- DHCP Snooping / IP Source Guard를 정리하려면 [[10_학습 노트/시스템보안/네트워크보안/Dynamic ARP Inspection|Dynamic ARP Inspection]]을 먼저 본다.
  - 이 노트 안에 DHCP Snooping Binding DB, DAI, IP Source Guard 관계가 이미 정리되어 있다.
  - 별도 stable note는 아직 없으므로 새 노트를 만들 때는 이 노트와 [[10_학습 노트/시스템보안/네트워크보안/ARP 스푸핑|ARP 스푸핑]]의 방어 표를 source로 삼는다.
- Wireshark 필터를 정리하려면 한 노트에서 시작하지 않는다.
  - HTTP/Form 관찰: [[10_학습 노트/시스템보안/네트워크보안/HTTP 로그인 평문 노출|HTTP 로그인 평문 노출]]
  - DNS Query/Response 관찰: [[10_학습 노트/시스템보안/네트워크보안/DNS 스푸핑 실습|DNS 스푸핑 실습]]
  - SSH 암호화 관찰: [[10_학습 노트/시스템보안/네트워크보안/SSH 암호화 패킷 관찰|SSH 암호화 패킷 관찰]]
  - Telnet 평문 관찰: [[10_학습 노트/시스템보안/네트워크보안/Telnet 평문 노출|Telnet 평문 노출]]
- MITM 실습 흐름을 다시 보면 [[10_학습 노트/시스템보안/네트워크보안/ARP 스푸핑|ARP 스푸핑]] → [[10_학습 노트/시스템보안/네트워크보안/DNS 스푸핑|DNS 스푸핑]] → [[10_학습 노트/시스템보안/네트워크보안/Ettercap Filter 패킷 변조 실습|Ettercap Filter 패킷 변조 실습]] → [[10_학습 노트/시스템보안/네트워크보안/SSH MITM 실습|SSH MITM 실습]] 순서로 본다.

## source / lab 경계

- 이 목차는 핵심 concept/lab 진입점만 둔다. 패킷 캡처, 긴 명령 출력, 실습 세부 증거는 각 lab note 본문에서 확인한다.
- `DHCP Snooping`, `IP Source Guard`, `Wireshark 필터`는 아직 독립 stable note가 아니다. 현재는 위 재시작 지점의 기존 노트를 source로 삼는다.
- `SSH MITM 실습`은 완료 lab note다. SSH 개념을 먼저 복구하려면 [[10_학습 노트/시스템보안/네트워크보안/SSH 보안 구조|SSH 보안 구조]]와 [[10_학습 노트/시스템보안/네트워크보안/SSH Downgrade Attack|SSH Downgrade Attack]]을 먼저 본다.
