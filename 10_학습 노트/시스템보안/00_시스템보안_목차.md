# 시스템보안 목차

## 개요

- [[서론(이라쓰고 빠르게 휘갈겨 쓴거)]]

시스템보안 영역은 현재 `네트워크보안`과 `웹보안`을 중심으로 정리한다. 운영체제 보안, 취약점 분석, 포렌식/로그 분석은 아직 별도 축으로 확장하기 전이다.

## 폴더 신설 후보

이번 변경에서는 폴더를 만들지 않는다. 아래 조건이 충족되면 하위 폴더 신설을 검토한다.

| 후보 | 조건 |
|---|---|
| `웹보안/SQL Injection/` | SQLi 방어 노트와 추가 실습까지 생겨 현재 웹보안 목차가 다시 비대해질 때 |
| `웹보안/실습/` | 웹보안 실습 노트가 개념 노트보다 많아져 실습 흐름만 따로 보고 싶을 때 |
| `취약점 분석/` | 웹 취약점뿐 아니라 시스템/서비스 취약점 노트가 함께 늘어날 때 |
| `포렌식-로그분석/` | 침해대응, 로그 분석, 증거 수집 노트가 독립 축으로 생길 때 |

## 운영 메모

- 네트워크보안과 웹보안 세부 노트는 하위 목차에서 관리한다.
- 파일 이동이나 폴더 신설은 먼저 목차가 감당하지 못하는지 확인한 뒤 진행한다.
- source-digest, PDF 구조 지도, raw 메모는 stable note가 아니다. 재시작할 때는 하위 MOC의 `현재 재시작 지점`을 먼저 확인한다.
