# 시스템보안 목차

## 개요

시스템보안 영역은 현재 `네트워크보안`과 `웹보안`을 중심으로 정리한다. 운영체제 보안, 취약점 분석, 포렌식/로그 분석은 아직 별도 안정 축이 없다.

## 하위 목차

- [[10_학습 노트/시스템보안/네트워크보안/00_네트워크보안_목차|네트워크보안 목차]] - ARP/DNS/SSH/MITM, 평문 통신, 패킷 관찰
- [[10_학습 노트/시스템보안/웹보안/00_웹보안_목차|웹보안 목차]] - HTTP, 인증/세션, XSS, CSRF, SQL Injection, File Upload

## 현재 재시작 지점

- 웹보안: `5-20_웹보안.pdf` p.1-136 source coverage는 한 바퀴 닫혔다. 복습과 후속 작업은 [[10_학습 노트/시스템보안/웹보안/00_웹보안_목차|웹보안 목차]]에서 시작한다.
  - Apache 차단 설정의 런타임 검증은 [[10_학습 노트/시스템보안/웹보안/웹쉘 Upload 및 실행 실습|웹쉘 Upload 및 실행 실습]]의 남은 확인이다.
- 네트워크보안: MITM 흐름과 프로토콜별 관찰 노트는 [[10_학습 노트/시스템보안/네트워크보안/00_네트워크보안_목차|네트워크보안 목차]]에서 복구한다.
  - DHCP Snooping / IP Source Guard는 [[10_학습 노트/시스템보안/네트워크보안/Dynamic ARP Inspection|Dynamic ARP Inspection]]의 Binding DB 관계 설명에서 이어간다.

## 영역 경계

- 네트워크 장비·토폴로지 학습과 공격·방어 실습을 섞지 않는다.
- 운영체제 보안, 취약점 분석, 포렌식/로그 분석은 실제 안정 노트와 재시작점이 생길 때 별도 축으로 확장한다.
