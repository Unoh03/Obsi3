# 실제 사례 후보

## 선별 기준

각 사례는 다음 질문에 구체적으로 답할 수 있어야 한다.

1. 어떤 상황과 문제가 있었는가?
2. 장운호가 직접 맡거나 주도한 행동은 무엇인가?
3. 도중에 어떤 판단이나 방향 전환을 했는가?
4. 결과와 검증 근거는 무엇인가?
5. 미완성·실패·팀 기여 등 밝혀야 할 한계는 무엇인가?
6. 지원 직무의 역량과 어떻게 연결되는가?
