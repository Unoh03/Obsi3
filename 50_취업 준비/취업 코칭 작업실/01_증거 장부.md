# 취업 코칭 증거 장부

## 목적과 범위

- 목적: 자기소개서와 면접에 사용할 주장을 실제 작업 증거와 연결한다.
- 포함: 사용자 진술, 채팅 기록, Vault 문서, 코드, Git 기록, 실행 결과, 보고서와 발표자료
- 제외: 출처 없는 미화, 팀 전체 성과를 개인 성과로 바꾸는 서술, 미완성 기능의 완성 주장

## 검증 메모

- 팀 성과와 개인 기여를 분리한다.
- 실행·검증 증거가 없더라도 문제 발견, 설계 변경, 중단 판단은 실제 사례가 될 수 있다.
- `D:/kisa-webapp-checker`는 2026-07-22 기준 `attack-active --validate-only`에서 18개 ready, 2개 mode skip을 출력했으며 실제 HTTP 요청은 보내지 않았다.
- `D:/kisa-webapp-checker`의 validate-only evidence는 `.gitignore`의 `evidence/*` 규칙에 따라 추적되지 않는다.
- 웹 통합 스크립트의 이번 검증은 Bash 문법 검사만 수행했다. 서버 런타임 성공을 새로 주장하지 않는다.
