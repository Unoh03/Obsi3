# 취업 코칭 증거 장부

## 목적과 범위

- 목적: 자기소개서와 면접에 사용할 주장을 실제 작업 증거와 연결한다.
- 포함: 사용자 진술, 채팅 기록, Vault 문서, 코드, Git 기록, 실행 결과, 보고서와 발표자료
- 제외: 출처 없는 미화, 팀 전체 성과를 개인 성과로 바꾸는 서술, 미완성 기능의 완성 주장

## 판정 기준

| 상태 | 의미 |
|---|---|
| 확인 | 사용자 진술 또는 현재 자료로 사실이 확인됨 |
| 보강 필요 | 사실이지만 자소서·면접용 세부 증거를 더 찾아야 함 |
| 범위 제한 | 계획·부분 구현·중단 등 한계를 함께 밝혀야 함 |
| 제외 | 본인의 실제 행동을 뒷받침하지 못함 |
