# 웹 서비스 보안 노트 작성 계획

## 1. 목적

이 문서는 PDF p.271부터 시작되는 **웹 서비스 보안 진단 항목**을 노트화하기 위한 작성 계획과 공통 템플릿을 정리한다.

기존 웹 애플리케이션 취약점 노트는 CARE 애플리케이션의 PHP 코드와 기능을 중심으로 분석했다. 반면 웹 서비스 보안 항목은 Apache 웹서버 설정, 파일 권한, 모듈, 헤더, 로그, SSL/TLS 등 서버 운영 설정을 중심
